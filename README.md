# How do I put more then one on a page
<p>If you want to use more then one on a page you will have to stylelive it like this: <pre>&lt;script&gt;
window.onkeypress = function(event) {
   if (event.keyCode == 56) {
// Your JS here
   }
   if (event.keyCode == 115) {
// Your JS here
   }
}
&lt;/script&gt;</pre></p>
